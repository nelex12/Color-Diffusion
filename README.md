Color Diffusion
Проект по диффузионной модели для колоризации

Основные файлы:
- BucketsCreation.py ищет в папке dataset подпапки train и val. На их основе генерирует .json файлы, который код в дальнейшем использует для обучения изображений на разных разрешениях
- Train.py — обучение модели. В самом простом случае, можно просто запустить файл после загрузки подходящего датасета и создания бакетов
- Dataset.py — работа с датасетом. Для использования не нужно
- Architecture.py — архитектура модели
- ScheduleAndSampleGen.py — свой простой вариант сэмплера. Можно использовать любые другие на практике


Сама архитектура основана на гибриде свёрток + трансформеров с самовниманием и кросс-вниманием. Имеет Unet подобный вид. Принимает серое изображение, зашумленное цветное, а также временной шаг t

Модель показала быстрое понимание основной структуры изображений, но для цвета требуется время.
Желательно увеличивать размер батча или накопления и использовать порядка 10 млн шагов оптимизации т.к. данных на входе очень много и они очень разнообразны. Модель плохо реагирует на шум в градиентах.
Сам код написал 1 человеком от скуки, поэтому претензий на особую чистоту, корректность или качество изначально не было
